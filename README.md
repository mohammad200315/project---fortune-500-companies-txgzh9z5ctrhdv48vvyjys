# Fortune 500 Analytics Dashboard

# The workflow includes:
. Data Cleaning & Preprocessing: Handling missing values, normalizing financial metrics, and preparing datasets for analysis.
. Exploratory Data Analysis (EDA) & Visualization: Trend analysis, industry comparison, revenue and profit visualizations using Plotly.
. Machine Learning Predictions: Forecasting 2024 revenue for top companies using regression models.
. Interactive Dashboard: Built with Streamlit for interactive exploration of historical data, predictions, and model performance.
. The dashboard provides insights into revenue trends, profit margins, company rankings, and industry analysis.
. Technologies used: Python, Pandas, NumPy, Plotly, Streamlit, Scikit-learn
